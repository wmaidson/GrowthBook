## Especificadores de formato em C

| Format Specifier	 | Type |
| ------ | ------ |
| %c |	Character |
| %s | String |
| %d | Signed integer |
| %f | Float values  |
| %l or %ld or %li | Long |
| %lf | Double |
| %n | Prints nothing |
